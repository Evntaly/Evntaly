# Evntaly Backend

NestJS API for the Evntaly event analytics platform.

See the [root README](../README.md) for full documentation, Docker setup, and API reference.

## Quick Start

```bash
# From repo root
cp .env.example .env
docker compose up -d
```

## Development

```bash
npm install
cp ../.env.example .env
npm run start:dev
```
